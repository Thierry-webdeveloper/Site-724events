Projet 9 de la formation **Intégrateur Web** (OpenClassrooms).

## 🎯 Description

L'application est le site d'une agence évènementielle.  
Reprise et finalisation du travail d'un développeur freelance ayant abandonné le projet.
Mission : identifier, corriger les bugs du site vitrine one-page de l'agence événementielle **724events**,
valider les corrections par les tests unitaires, et rédiger un cahier de recette complet.

## 🐛 Bugs corrigés

- **Slider** — Suppression de l'image blanche et correction de l'ordre d'affichage (du plus ancien au plus récent)
- **Filtre Événements** — Le filtre par catégorie est opérationnel et les mois s'affichent correctement
- **Formulaire de contact** — Le message de confirmation apparaît après l'envoi
- **Footer** — L'image de la dernière réalisation s'affiche correctement

## 🧪 Tests

L'application est testée avec **Jest** et **React Testing Library**.

```bash
# Lancer tous les tests
yarn test

# Lancer les tests en mode watch
yarn test --watch
```

## 🚀 Installation et démarrage

```bash
# Cloner le repository
git clone https://github.com/Thierry-webdeveloper/724events.git

# Installer les dépendances
yarn install

# Lancer l'application en mode développement
yarn start
```

Le site est accessible sur [http://localhost:3000](http://localhost:3000).

## 🛠️ Stack technique
